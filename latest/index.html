IntervalMatrices.jl

IntervalMatrices is a Julia package to work with matrices that have uncertain parameters.

Features

  • An IntervalMatrix type.

  • Arithmetics between scalars, closed intervals and interval matrices.

  • Overapproximation and underapproximation routines for the exponential of an interval matrix at different orders.
